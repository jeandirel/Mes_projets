DE DEMANDE DE
BOURSE EN LIGNELes étapes a suivre

Etape 1 : Activation du compte


1. Se connecter sur www.anbg.online pour
accéder a eBourse ;

2. Cliquer sur « je n'ai pas de compte, je crée
mon compte » ;

2. Cocher « Bachelier » ;

3. Cliquer sur « Continuer » ;

4. Cocher « Baccalauréat Gabonais » ;

5. Cliquer sur « Continuer » ;

6.Renseigner votre numéro de baccalauréat ;
Inscrire votre date de naissance au format
jj/mm/aaaa ;

7. Cliquer sur « Je crée mon compte » ;

8. Saisir votre moyenne annuelle de terminale ;
Saisir votre adresse mail (cette adresse mail
doit étre personnelle, valide et accessible) ;
Cliquer sur « Envoyer mon code » ;

9. Récupérer votre code de validation dans votre
boite mail ;

10. Insérer votre code de validation ;

11. Cliquer sur « valider I'inscription » ;

12. Compléter le « formulaire de création du
compte » en renseignant les zones de texte
encadrées en rouge, mettre les informations y
relatives, puis cliquer sur « Suivant » ;

14. Remplir le formulaire de « renseignements
sur les parents d'éléves », puis cliquer sur
« suivant ». Votre espace eBourse est
désormais ouvert.



Etape 2 : Téléversement des pieces

d’eligibilite

Cliquer sur l'onglet « documents administratifs » ;
Cliquer sur |'onglet « Téléverser une piéce » ;

Téléverser les documents obligatoires exigés par le
systeme, a savoir:

01 photo d'identité sur fond blanc (toute photo non
conforme sera systématiquement rejetée) ;

01 copie certifiée conforme de l'acte de naissance
gabonais ;

01 copie certifiée conforme du certificat de nationalité
et jugement déclaratif du tribunal ou décret de
naturalisation pour tous ceux ayant acquis la
nationalité gabonaise.

Etape 3 : Demande de bourse

1. Accéder au site www.anbg.online pour se
connecter a eBourse ;

2. Saisir votre identifiant (adresse mail ou
numéro de téléphone) ;

3. Entrer votre mot de passe ;

4. Cliquer sur « Se connecter ». Votre espace
eBourse est ouvert ;

5. Cliquer sur « Nouvelle demande » ;

6. Renseigner le pays ol vous souhaitez étudier
au titre de l'année académique 2024-2025 ;

7. Indiquer la discipline ;

8. Sélectionner l'établissement ;
9. Sélectionner le dipléme ;

10. Saisir le coat de la formation ;

11. Cliquer sur « Enregistrer » pour finaliser votre
demande.

A la suite de cette action, l'apprenant recoit dans
sa boite mail une notification de félicitations. Le
dossier a été créé.
